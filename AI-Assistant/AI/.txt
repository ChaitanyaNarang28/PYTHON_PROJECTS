```python
def print_star_pattern(rows):
  """
  Prints a star pattern based on the given number of rows.

  Args:
    rows: The number of rows in the pattern.
  """

  # Upper half of the pattern
  for i in range(1, rows + 1):
    print("*" * i)

  # Lower half of the pattern
  for i in range(rows - 1, 0, -1):
    print("*" * i)

# Get the number of rows from the user
rows = int(input("Enter the number of rows: "))

# Print the star pattern
print_star_pattern(rows)
```

**Explanation:**

1. **`print_star_pattern(rows)` Function:**
   - Takes `rows` as input to determine the size of the pattern.
   - Uses two `for` loops to create the upper and lower halves of the pattern.

2. **Upper Half (Ascending):**
   - `for i in range(1, rows + 1):` iterates from 1 to `rows` (inclusive).
   - `print("*" * i)` prints `i` number of stars in each row, increasing the number of stars with each iteration.

3. **Lower Half (Descending):**
   - `for i in range(rows - 1, 0, -1):` iterates from `rows - 1` down to 1, decreasing the number of stars in each row.
   - `print("*" * i)` prints `i` number of stars in each row.

4. **User Input:**
   - `rows = int(input("Enter the number of rows: "))` prompts the user to enter the number of rows.

5. **Function Call:**
   - `print_star_pattern(rows)` calls the function to generate and print the star pattern.

**Example Output:**

If the user enters `5` as the number of rows, the output will be:

```
*
**
***
****
*****
****

**
*
```
